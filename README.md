## 简介

Server Sentinel 是一款 SJMCL 扩展，在启动器内实时监控 Minecraft 服务器的运行状态，包括在线人数、延迟、TPS 等关键指标。

提供两个版本：
- **英文版**（org.yoshino.server_sentinel）
- **中文版**（org.yoshino.server_sentinel.zh）

## 功能特色

- 📊 实时显示服务器在线人数与最大玩家数
- ⏱ 颜色分级延迟监控（绿/黄/橙/红）
- 📈 实时 TPS 监控
- ➕ 支持添加多个服务器

## 使用方法

1. 从 Releases 下载对应的 .sjmclx 文件
2. 打开 SJMCL → 设置 → 扩展管理 → 导入
3. 在扩展设置页添加要监控的服务器地址

## 兼容性

- 最低启动器版本: 1.1.3
